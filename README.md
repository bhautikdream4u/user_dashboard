# User Dashboard WPS247

A custom WordPress plugin that provides a frontend user dashboard for managing user accounts, profiles, memberships, and personalized user features without requiring access to the WordPress admin area.

## Features

- Frontend User Dashboard
- User Profile Management
- Custom Dashboard Pages
- Membership & Subscription Support
- User-Specific Content Display
- Responsive Design
- WordPress Coding Standards
- Extensible and Developer Friendly

## Requirements

- WordPress 6.0+
- PHP 8.0+
- MySQL 5.7+ or MariaDB equivalent

## Installation

1. Download the plugin ZIP file.
2. Go to **WordPress Admin → Plugins → Add New**.
3. Click **Upload Plugin**.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure settings as required.

## Usage

After activation:

1. Create a page for the user dashboard.
2. Add the provided shortcode or block (if applicable).
3. Assign appropriate user permissions.
4. Customize dashboard sections as needed.
